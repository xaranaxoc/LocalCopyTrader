# FTH Trade Copier

Локальный копитрейдер для MetaTrader 5 — копирует сделки из мастер-терминала в слейв-терминалы на одном компьютере.

## Быстрый старт

1. Скачайте `FTHTradeCopier.exe` из [последнего релиза](https://github.com/xaranaxoc/LocalCopyTrader/releases)
2. Запустите двойным кликом — Python не нужен
3. Добавьте мастер-терминал и слейв-терминалы
4. Нажмите **▶ Старт**

> Настройки сохраняются в `%APPDATA%\MT5CopyTrader\`

## Настройка

### Мастер-терминал

- Нажмите **Обзор** рядом с полем МАСТЕР и выберите `terminal64.exe`
- Обычные пути:
  - `C:\Program Files\MetaTrader 5\terminal64.exe`
  - `C:\Users\ИМЯ\AppData\Roaming\MetaQuotes\Terminal\ХЭШ\terminal64.exe`

### Слейв-терминалы

Нажмите **+ Добавить** и заполните:

| Поле | Описание |
|------|----------|
| **Имя** | Название для отображения |
| **Путь** | Путь к `terminal64.exe` слейва |
| **Символы** | Пары мастер→слейв (например `XAUUSD → XAUUSDb`) |
| **Тип риска** | `% от баланса` или `Фикс. сумма $` |
| **Значение риска** | `1.0` (1%) или `100` ($100) |
| **Лот если нет SL** | Лот по умолчанию при отсутствии стоп-лосса |

## Расчёт лота

Используется метод PositionSizer:

```
Расстояние до SL  = |цена открытия − стоп-лосс|
Количество тиков  = Расстояние / размер тика
Стоимость тика    = contract_size × tick_size × rate(валюта_прибыли → валюта_депозита)
Сумма риска       = Баланс × Риск% / 100  (или фиксированная сумма)
Лот               = Сумма риска / (Количество тиков × стоимость тика)
```

Работает корректно для любых инструментов: форекс, индексы, commodities, JPY-пары.

## Что копируется

| Событие мастера | Действие на слейве |
|---|---|
| Открытие позиции | Открыть с рассчитанным лотом |
| Закрытие позиции | Закрыть |
| Новый отложенный ордер | Разместить |
| Отмена ордера | Отменить |
| Изменение SL/TP | Обновить пропорционально |

SL и TP рассчитываются как процент от цены открытия — корректно для разных брокеров с разными ценами.

## Маппинг символов

Копируются только символы из настроек слейва. Примеры:

- `XAUUSD → XAUUSD` — одинаковый символ
- `XAUUSD → GOLD` — разное название у брокера
- `EURUSD → EURUSDb` — суффикс брокера

Автоматическое сопоставление: регистронезависимое + CFD-алиасы + брокерные суффиксы.

## Важные замечания

- **AutoTrading** должен быть включён на слейв-терминалах (⚠AT в интерфейсе = выключен)
- **Терминал должен быть запущен** — иначе сделка пропускается
- **Один аккаунт на терминал** — для нескольких аккаунтов установите MT5 в разные папки
- **Не удаляйте `state.json`** пока копитрейдер работает — это приведёт к дублированию сделок
- Скопированные сделки имеют комментарий `CT_ТИКЕТ`
