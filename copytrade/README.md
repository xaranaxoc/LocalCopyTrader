# MT5 Copy Trader

Локальный копитрейдер для MetaTrader 5 — копирует сделки из одного терминала в другие на одном компьютере.

---

## Быстрый старт

### Вариант A: Запуск EXE (рекомендуется, Python не нужен)

1. Скачайте `MT5CopyTrader.exe` из папки `dist\`
2. Положите его в любую папку
3. Запустите двойным кликом

> **Примечание:** `config.json` и `state.json` создаются автоматически рядом с EXE.

---

### Вариант B: Сборка EXE самостоятельно

**Требования:** Python 3.8+ установлен и добавлен в PATH

1. Откройте папку `copytrade\` в проводнике
2. Запустите `build.bat` двойным кликом
3. Дождитесь завершения (~1-2 минуты)
4. Готовый файл: `dist\MT5CopyTrader.exe`

---

## Настройка

### 1. Мастер-терминал

- Запустите MT5 мастера и войдите в нужный аккаунт
- В приложении нажмите **Обзор** рядом с полем МАСТЕР
- Выберите `terminal64.exe` мастер-терминала

Обычные пути к терминалу:
```
C:\Program Files\MetaTrader 5\terminal64.exe
C:\Users\ИМЯ\AppData\Roaming\MetaQuotes\Terminal\ХЭШ\terminal64.exe
```

### 2. Слейв-терминалы

- Нажмите **+ Добавить** в секции СЛЕЙВЫ
- Заполните настройки слейва:

| Поле | Описание |
|------|----------|
| **Имя** | Любое название для отображения |
| **Путь** | Путь к `terminal64.exe` слейва |
| **Символы** | Пары мастер→слейв (например `XAUUSD → XAUUSD`) |
| **Тип риска** | `% от баланса` или `Фикс. сумма $` |
| **Значение риска** | Например `1.0` (1%) или `100` ($100) |
| **Лот если нет SL** | Лот по умолчанию, если у сделки нет стоп-лосса |

### 3. Запуск копирования

- Убедитесь, что все нужные терминалы MT5 **запущены и залогинены**
- Нажмите **▶ Старт**
- Статус каждого терминала отображается в секции СТАТУС
- Все события отображаются в секции ЛОГ

---

## Как работает расчёт лота

Используется метод **PositionSizer** (как на EarnForex):

```
Расстояние до SL = |цена открытия − стоп-лосс|
Количество тиков = Расстояние / размер тика
Сумма риска      = Баланс × Риск% / 100  (или фиксированная сумма)
Лот              = Сумма риска / (Количество тиков × стоимость тика)
```

Если у сделки мастера **нет стоп-лосса** → используется **Лот если нет SL**.

## Что копируется

| Событие на мастере | Действие на слейве |
|---|---|
| Открытие BUY/SELL | Открыть позицию с рассчитанным лотом |
| Закрытие позиции | Закрыть соответствующую позицию |
| Новый отложенный ордер | Разместить такой же ордер |
| Отмена отложенного ордера | Отменить ордер |
| Изменение SL/TP | **Не копируется** (только при открытии) |

**SL и TP** копируются только в момент открытия сделки.  
Если символы мастера и слейва **разные** (например `XAUUSD → GOLD`) — SL/TP не копируются.

---

## Маппинг символов

Копируются **только** символы, указанные в настройках слейва.

Примеры:
- `XAUUSD → XAUUSD` — одинаковый символ
- `XAUUSD → GOLD` — разные названия одного инструмента у разных брокеров
- `EURUSD → EURUSD.m` — суффикс брокера

---

## Важные замечания

- **Терминал должен быть запущен** — если слейв-терминал не запущен, сделка на него не копируется (без ошибки, просто пропускается)
- **Один аккаунт на терминал** — используется тот аккаунт, который сейчас залогинен
- **Несколько копий MT5** — для нескольких аккаунтов нужно установить MT5 в разные папки
- **Комментарий к сделкам** — все скопированные сделки получают комментарий `CT_ТИКЕТ` для идентификации

## Файлы

| Файл | Описание |
|------|----------|
| `config.json` | Настройки (создаётся автоматически) |
| `state.json` | Маппинг тикетов (создаётся автоматически) |

## Установка нескольких MT5 на одном компьютере

1. Скачайте установщик MT5 с сайта брокера
2. При установке выберите **другую папку** (например `C:\MT5_Account2\`)
3. Запустите новый терминал и войдите в нужный аккаунт
4. Укажите путь к этому терминалу в настройках слейва
